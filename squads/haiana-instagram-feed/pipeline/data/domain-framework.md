# Domain Framework

## Fluxo operacional recomendado

1. Definir o foco clínico e a intenção do post.
2. Delimitar a pergunta real do público.
3. Verificar a base técnica mínima.
4. Traduzir o tema em uma promessa editorial segura.
5. Escolher um formato de carrossel.
6. Estruturar a progressão do carrossel.
7. Ajustar o texto para linguagem simples.
8. Aplicar o branding.
9. Inserir salvaguardas clínicas.
10. Revisar para Instagram.

## Critérios de decisão

- Use `Mito vs Realidade` quando o problema principal for crença errada ou confusão comum.
- Use `Lista` quando o usuário precise guardar a informação para consulta futura.
- Use `Tutorial` quando houver uma sequência prática de ações.
- Use `Problema -> Solução` quando o tema pedir identificação rápida com dor e alívio.
- Se o tema envolver medicação, incluir sempre contexto, limites e chamada para individualização clínica.
- Se o tema puder gerar culpa ou vergonha, preferir tom acolhedor e explicativo.
